# Contribución
- Ramas: `feature/<nombre>`, `fix/<nombre>`, `chore/<nombre>`
- Merge a `main` solo vía PR con checklist DoD

## Definition of Done (DoD)
- UI responsiva y navegable
- Datos mock funcionales
- 1 caso de prueba documentado verifica el criterio de aceptación
- Sin errores de consola
- Evidencias (SS-XX / LOG-XX) en `docs/`

## Conventional Commits
- `feat:`, `fix:`, `docs:`, `chore:`, `refactor:`, `test:`
- Ej.: `feat(timeline): mostrar etapas con fotos y fecha`
